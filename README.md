# Credit Card Financial Dashboard

## Project Overview
This project focuses on developing an interactive Credit Card Financial Dashboard using Power BI, SQL, and DAX. The dashboard provides insights into revenue, transactions, customer behavior, card performance, and overall business trends. It helps stakeholders monitor key performance indicators and make data-driven decisions.

## Tools & Technologies
- Power BI
- SQL
- DAX
- Excel / CSV

## Project Workflow
- Imported customer and transaction datasets into SQL Database.
- Performed data cleaning and transformation.
- Connected SQL database with Power BI.
- Created calculated columns and measures using DAX.
- Designed interactive dashboards and KPI visualizations.
- Generated business insights through data analysis.

## Key KPIs
- Total Revenue
- Total Transaction Amount
- Total Transaction Count
- Total Interest Earned
- Customer Count
- Activation Rate
- Delinquency Rate

## DAX Calculations
- Revenue Calculation
- Current Week Revenue
- Previous Week Revenue
- Week-over-Week (WoW) Growth
- Customer Age Group Segmentation
- Income Group Classification

## Dashboard Features
- Interactive Filters and Slicers
- Dynamic KPI Cards
- Weekly Revenue Tracking
- Customer Segmentation Analysis
- Card Category Performance Analysis
- State-wise Performance Monitoring
- Trend Analysis and Business Insights

## Key Insights
- Weekly revenue and transaction performance tracking.
- Revenue contribution analysis by card category.
- Customer segmentation based on age and income groups.
- Gender-wise revenue comparison.
- State-wise contribution analysis.
- Monitoring activation and delinquency rates.

## Project Outcome
Developed a business intelligence solution that transforms raw customer and transaction data into actionable insights. The dashboard enables effective monitoring of credit card operations, customer behavior, and revenue performance.
